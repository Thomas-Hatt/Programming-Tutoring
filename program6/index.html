Program 6


Task: Susan runs several bowling tournaments and needs a program to calculate the average for individual bowlers. She enters the scores in a file and would like a program to add up the scores and divide by the total. Susan would also like the program to have the option of displaying the scores to screen in sequential order.

Your task is to create a program that displays the bowling scores to screen and calculate the bowler’s average.

Input:
  1. The user enters a letter to go to a specific function
    • C: Calculate the bowler's average score
      • Attempts to read the scores from a pre-defined file
    • D: Display the the current scores
    • X: Exit application
Process:
    Program written in Python containing:
  1. A main function that displays a menu of choices.
  2. A function named ShowScores that displays scores for all items in the text file provided. Each score should have a line number to the left starting at 1. For example: 1: 190 2: 247. (see output example below). This function accepts zero arguments and does not return any values. This function does read from a text file, so it needs error handling for IOErrors.
  3. A function named CalculateAvg that reads the text file provided, accumulates a total and divides the total by the number of items in the file provided. This function accepts zero arguments and returns a single value. The value returned is the bowler’s average as a result. This function does read from a text file, so it needs error handling for IOErrors and ValueErrors.
  4. Program is saved as Program6.py
Output:
  1. Scenario 1 (User enters letter C):
    Calculates the average score from the 'games.txt' file and returns the average score in a float, or nothing if there was an error.
  2. Scenario 2 (User enters letter D):
    Iterates through the entire file and displays them on screen.
  3. Scenario 3 (User enters letter X):
    The program prints a goodbye message and closes.

The formatted class average will return 203.96 based on the text file provided to you. You must develop your program to count the lines in the event the text file were to change.
